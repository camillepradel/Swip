public class WebAppResourceConfig
extends ScanningResourceConfig

A mutable implementation of DefaultResourceConfig that dynamically searches for root resource and provider classes in the Web application resource paths declared by the property ClasspathResourceConfig.PROPERTY_CLASSPATH. If that property is not included in the map of initial properties passed to the constructor then the Web application paths "WEB-INF/lib" and "WEB-INF/classes" are utlized.
